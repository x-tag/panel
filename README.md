# Dev Setup

```

$ npm install

$ bower install

$ node demo.js   

// Navigate to localhost:3000/demo/demo.html

```


## Syntax


```
	<x-panel src="some-page.html"> </x-panel>
```

## Usage

```
	var panel = document.createElement('x-panel');
	// panel will get the content as soon as the src property is set
	panel.src = "page2.html";  

```
